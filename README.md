# 🚗 Dashboard de Vendas Porsche

Projeto desenvolvido como parte de estudos e portfólio, com foco em análise e visualização de dados de vendas.

---

## 📋 Sobre o Projeto
Interface interativa para acompanhar os dados de vendas da marca Porsche, com design alinhado à identidade visual da marca e funcionalidades práticas para análise.

## ✨ Funcionalidades
- ✅ Filtros dinâmicos: Modelo, Ano, Cidade e Forma de Pagamento
- 📊 Indicadores chave (KPIs): Total de vendas, faturamento, modelo e cidade mais vendidos
- 📈 Gráficos interativos: Distribuição por cidade, ano e forma de pagamento
- 📋 Tabela completa com todos os registros de vendas
- 💡 Insights automáticos por região

## 🛠️ Tecnologias Utilizadas
- **HTML5** → Estrutura da página
- **Tailwind CSS** → Estilização e design responsivo
- **JavaScript** → Lógica, filtros e manipulação dos dados
- **Chart.js** → Criação dos gráficos dinâmicos
